# Console Service

## Built with
* Scala 2.10
* Scalatest
* Specs2
* JUnit



## IDE integration

The resultant project contains both Eclipse and Intellij plugins to
auto generate projects.

## usage

* [Install g8](http://github.com/n8han/giter8#readme)
* from the shell run "g8 crispwalrus/service
* cd ${app.name}
* sbt compile
